Examples of SAE's




Example 1: Producing for a grocery store


Producing for a grocery store by farming crops is a big benefit for the community, and can give you a lot of service hours if you're not being paid. Resources you need are farming tools(shovel, hoe, etc.), watering can, seeds, and you need enough space for farmland.



Example 2: Programming an SAE research site(My SAE you are currently viewing now)


Programming an SAE research page is in a technologic level of SAE. All you need is a broadband internet connection, a tool for coding(One that supports HTML), and basic HTML skills. HTML is a code language used to make websites, and intergrates website info in apps programmed in other languages. When programming an SAE research site, be sure to add examples and explain the different types of SAEs.



Example 3: Breeding animals


Breeding animals is very important, because it's how more animals come to the world for a better reproduction of food, making it a very big help to the community in need of more food.
